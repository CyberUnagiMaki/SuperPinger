# SuperPinger

Modern Android networking utility for ping, port scanning and host monitoring.

SuperPinger is a powerful network diagnostic tool for Android. Whether you're a network engineer, system administrator, homelab enthusiast or simply checking your internet connection — SuperPinger provides fast and convenient tools for network analysis directly from your phone.

---

## Features

### 🔹 PING
Check response time for any IP address or domain.

- Real-time latency graph
- Packet statistics (min / avg / max / loss)
- Per-packet results
- Unlimited auto ping mode
- Custom packet count
- Adjustable timeout, interval and packet size

---

### 🔹 DNS LOOKUP
Instantly resolve domains and IP addresses.

- Forward DNS lookup (domain → IP)
- Reverse DNS lookup (IP → hostname)
- A and AAAA records support

---

### 🔹 PING HISTORY
All ping sessions are automatically saved locally.

- Session history
- CSV export support
- Easy access to previous diagnostics
